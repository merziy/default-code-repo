The CSS Property Display: A guide

This property determines whether an element is treated as a block or inline. This also impacts the layout its children are displayed as; some examples are flow, flex, and grid layouts. Some potential values are fully defined in their own specifications.

Possible values for Display:
display: block; Displays an element as a block element (i.e., <h3>)
display: inline; Displays an element as an inline element (i.e., <div>)
display: flex; Displays elements as a two axis, one-dimensional layout model. See this guide for more details.
display: grid; Displays elements as a block-level grid container. See this guide for more details.
display: contents; Hides the containers, therefore making child elements children of the element in the DOM that is one node above.
